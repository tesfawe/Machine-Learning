# IMDB Movie Review Sentiment Classification

## Overview  
Sentiment analysis is a crucial tool for understanding user opinions from large-scale textual data. This project focuses on **classifying IMDB movie reviews** as **positive** or **negative** using **machine learning models**.  

With an increasing volume of **online user-generated reviews**, automated sentiment classification helps businesses and consumers quickly assess general opinions about movies.

## Dataset  
- **IMDB Movie Reviews Dataset**  
- Contains **50,000** reviews (balanced: **25,000 positive** & **25,000 negative**).
- Preprocessed textual data for improved classification.
